# [茴香豆](https://github.com/InternLM/HuixiangDou)

“茴香豆”是一个基于 LLM 的领域知识助手。特点：

1. 应对群聊这类复杂场景，解答用户问题的同时，不会消息泛滥
2. 提出一套解答技术问题的算法 pipeline
3. 部署成本低，见[技术报告 arxiv2401.08772](https://arxiv.org/abs/2401.08772)
